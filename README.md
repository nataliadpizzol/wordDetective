# How to Run:
1. Download the folder "Nano 1"
2. Open a terminal zsh terminal outside of xcode
3. Open the "Nano 1" folder directory on the terminal
4. Run the commands:
  ```shell
   $ swiftc *.swift -o wordDetective
  ```
  ```shell
  $ ./wordDetective
  ```
